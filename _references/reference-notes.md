## Layout and tone reference

- **Purpose:** Reference only — structure, whitespace, calm academic tone, and researcher-oriented credibility. Content, images, and design were **not** copied.

## Observed patterns (for design direction)

| Aspect | Reference approach | Applied in this site |
|--------|-------------------|----------------------|
| Navigation | English labels, horizontal | Same (HOME, RESEARCH, …) |
| Hero | Mission statement + affiliation | Catch copy + lab name + affiliation |
| News | Dated list on home | News section + dedicated NEWS page |
| Tone | Formal, research-focused | White background, blue accent, wide margins |
| Language | Primarily Japanese | JP/EN toggle with `data-ja` / `data-en` |
